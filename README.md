# Stagehand + Cerebras Problem Shooting

A repository for troubleshooting issues with Stagehand and Cerebras AI integration. This project showcases and helps resolve problems that occur when using Cerebras models with large web pages.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Create a `.env` file in the root directory with. Runs locally:
   ```
   CEREBRAS_API_KEY=your_cerebras_api_key_here
   ```

## Running the Project

Start the application:
```bash
npm run start
```

## Configuration

The project is configured to use Cerebras AI's API with the `llama-3.3-70b` model.
